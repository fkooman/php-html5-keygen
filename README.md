# Introduction

Simple HTML `<keygen>` demo.

# Prepare

You need (Composer)[https://getcomposer.org].

    $ git clone https://github.com/fkooman/php-html5-keygen.git
    $ cd php-html5-keygen
    $ composer install
    $ mkdir data
    $ php initCa.php

# Run

    $ php -S localhost:8080

# Use

Go to `http://localhost:8080/keygen.html`.
